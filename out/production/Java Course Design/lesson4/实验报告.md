#### 1.2.1 解释finally关键字的作用
```
finally关键字用于try后面，finally块中的代码总是执行，不论是否发生异常。一般用于清理工作、关闭链接等类型的语句。
```
#### 此程序中引发异常的语句是：
```Java
System.out.println("相除结果为：" + no1 / no2);
```
